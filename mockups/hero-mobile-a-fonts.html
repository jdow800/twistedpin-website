What's different from visual-reference.html Mobile A.

  • Display swapped from Anton (gothic, single-weight, Bebas-family) to Barlow Condensed 900 — humanist condensed, closer Proxima silhouette. Counters are squarer, terminals less aggressive, weight reads slightly heavier at the same px.
  • UI swapped from Inter (neo-grotesque, screen-optimised) to Montserrat (geometric humanist) — closer Proxima Nova proportion, especially on the eyebrow tracking.
  • Body kept on Roboto Slab — no meaningfully closer free Yorkten alt. Production swap is the obvious lift.
  • Added a phone-frame shell and small top-row chrome (brand mark + Menu) so it reads as a screen instead of a swatch — closer to how the hero will actually feel in the first viewport.
What to react to. Display personality (does Barlow Black at 64px in the hero feel like signage the way you want, or too narrow / too wide?). The eyebrow tracking on Montserrat. Whether "The bar that bowls." at this size + this face still lands as confident-not-shouty. The headline-to-subhead size ratio (64 / 15) — currently aggressive, can dial down.
